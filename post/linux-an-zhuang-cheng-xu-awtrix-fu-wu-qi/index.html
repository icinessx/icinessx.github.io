Linux安装程序AWTRIX服务器

在SSH终端中输入以下命令以自动安装

wget -N https://blueforcer.de/awtrix/awtrix.sh ; sudo sh awtrix.sh

[^启动后不久，即可通过http：// awtrix_ip：7000访问Web界面。]:

启动AWTRIX

sudo service awtrix start

停止AWTRIX

sudo service awtrix stop

AWTRIX重新启动

sudo service awtrix restart
